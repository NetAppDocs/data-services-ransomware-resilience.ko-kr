=== 사용자 행동 알림에서 파일 경로 제외

Ransomware Resilience는 사용자 동작 모니터링 경고에서 특정 파일 경로를 제외하는 기능을 지원합니다. 위험하지 않다고 판단되거나, 경고를 유발할 수 있는 작업을 볼륨 내 파일에서 수행하는 경우 해당 파일 경로를 제외할 수 있습니다.

자세한 내용은 link:https://docs.netapp.com/us-en/data-services-ransomware-resilience/task-users-manage.html["알림에서 파일 경로 제외"^]을 참조하십시오.
