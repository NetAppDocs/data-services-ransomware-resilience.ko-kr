=== 사용자 활동 알림에 대한 자동 응답

Ransomware Resilience는 이제 사용자 활동 경고가 발생할 때 스냅샷 생성 및 사용자 차단 자동화를 지원합니다. 자동화된 대응 기능을 통해 경고가 생성되는 즉시 사용자를 차단하고 복구용 스냅샷을 생성하여 수동 개입 없이 신속하게 대응할 수 있습니다.

자세한 내용은 link:https://docs.netapp.com/us-en/data-services-ransomware-resilience/task-automate-user-activity.html["사용자 활동 알림에 대한 응답 자동화"^]을 참조하십시오.
