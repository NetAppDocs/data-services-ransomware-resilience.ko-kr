백업 및 복구와 랜섬웨어 복원력을 모두 사용하는 경우, 두 제품으로 보호되는 공통 데이터는 랜섬웨어 복원력에 의해서만 청구됩니다.

워크로드가 보호됨으로 분류되면 구매한 용량 또는 종량제 구독에 포함됩니다. Ransomware Resilience는 하나 이상의 스냅샷 또는 백업 정책과 함께 탐지 정책이 활성화된 경우 워크로드를 보호됨으로 분류합니다. 탐지 정책은 있지만 백업 또는 스냅샷 정책이 _없는_ 워크로드는 위험으로 분류됩니다. 위험 워크로드는 구매한 용량에 포함되지 않습니다.

보호 대상 워크로드는 무료 평가 기간 종료 후 구매 용량 또는 구독료에 포함됩니다. Ransomware Resilience는 효율성 적용 전, 보호 대상 워크로드와 관련된 데이터(ONTAP에서 보고된 데이터 기준)를 GB 단위로 청구됩니다.

크기를 고려할 때 라이센스 비용은 액티브 파일 시스템의 논리적 용량을 기준으로 합니다(즉, 스냅샷이 사용하는 공간은 포함되지 않음).
